PROJECT OVERVIEW
================================================================================

Project Name: RealEstate Explorer / RealVest
Framework: Next.js 15 (React 18 with TypeScript)
Backend Services: Supabase, Sanity CMS
AI Integration: Google Genkit with Gemini 2.0 Flash
State Management: Zustand

This is a comprehensive real estate investment platform that allows users to 
browse properties, invest in real estate through various tiers, manage crypto 
assets, track investments through personalized dashboards, and access AI-powered 
property recommendations. The platform includes an admin panel for managing 
properties, users, and investment data.

15. AI-POWERED FEATURES
-----------------------------------
Purpose: Provide intelligent recommendations
Location: src/ai/flows/suggest-similar-properties.ts
Tech Stack: Google Genkit + Gemini 2.0 Flash

AI Flow: suggestSimilarProperties
Input:
- propertyDetails object:
  * propertyType: Type of property
  * location: Location/address
  * bedrooms: Number of bedrooms
  * bathrooms: Number of bathrooms
  * squareFootage: Area in sqft
  * price: Property price
  * amenities: Comma-separated amenities
  * description: Property description
  
- numberOfSuggestions: How many to recommend (default: 3)

Output:
- Array of similar properties with:
  * propertyType, location, bedrooms, bathrooms
  * squareFootage, price, amenities, description
  * similarityScore: 0-1 rating of similarity (1 = exact match)

Process:
1. Take property details
2. Send to Gemini 2.0 Flash model via Genkit
3. AI analyzes features and finds patterns
4. Returns ranked similar properties
5. Display on property details page

Genkit Configuration:
- Model: googleai/gemini-2.0-flash
- Plugin: Google AI (@genkit-ai/googleai)
- Framework: @genkit-ai/next for Next.js integration


================================================================================
                        DATA ARCHITECTURE & STORAGE
================================================================================

SUPABASE (PostgreSQL Database)
-----------------------------------
Primary database for user accounts, transactions, and investments

Key Tables:
1. profiles
   - id (PK)
   - email
   - full_name
   - created_at
   - updated_at

2. investments
   - id (PK)
   - user_id (FK → profiles)
   - property_id (from Sanity)
   - investment_amount
   - expected_roi
   - status
   - created_at
   - maturity_date

3. transactions
   - id (PK)
   - user_id (FK → profiles)
   - investment_id (FK → investments)
   - amount
   - transaction_type (enum: deposit, withdrawal, investment)
   - status (enum: pending, completed, failed)
   - created_at
   - payment_method

4. admin_controls
   - id (PK)
   - investment_growth_mode
   - roi_adjustment_rate
   - last_applied

Authentication:
- Managed through Supabase Auth
- Email/password authentication
- Session tokens
- User metadata storage


SANITY CMS (Content Management)
-----------------------------------
Headless CMS for property, agent, crypto, plan, and article content

Collections:
1. properties
   - _id (unique)
   - title, price, address
   - bedrooms, bathrooms, area
   - mainImage, gallery
   - isFeatured, propertyType
   - description, amenities
   - agentId (reference)

2. agents
   - _id (unique)
   - name, title
   - email, phoneNumber
   - profilePhoto
   - numberOfProperties

3. cryptoAssets
   - _id (unique)
   - symbol, name
   - price, change24h
   - expectedROI
   - description

4. plans
   - _id (unique)
   - name, price, priceRange
   - features (array)
   - popular (boolean)

5. articles
   - _id (unique)
   - title, description
   - link, dateCreated
   - mainImage

   - Manage properties

Security Features:
- Two-Factor Authentication (optional, available in dashboard)
- Session management with token expiration
- HTTPS encryption for all data transfer
- Protected API endpoints
- Input validation on all forms
- CSRF protection via Next.js
- XSS prevention with React's built-in escaping

Password Requirements:
- Minimum 8 characters

Data Protection:

                        CONCLUSION
================================================================================

This real estate investment platform is a comprehensive web application that 
combines property browsing, investment management, cryptocurrency assets, and 
AI-powered recommendations. It features a robust admin panel for platform 
management and secure user authentication.

The architecture leverages:
- Next.js for fast, server-side rendering
- Supabase for secure user management and financial data
- Sanity CMS for flexible content management
- Google Genkit with Gemini AI for intelligent recommendations

Key focus areas are security, scalability, user experience, and future 
extensibility. The platform is designed to grow and adapt to new features 
and business requirements while maintaining code quality and performance.
